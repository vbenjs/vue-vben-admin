// turbo-all

这个规则文件用于全局约束 AI 助手 (Antigravity) 的行为模式，确保在任何时候都最大程度减少对用户的人工干预和弹窗打扰:

1. **绝对优先使用静默原生工具**:
   - 在进行文件搜索、代码读取、文本过滤时，强制使用内置的 `grep_search`、`view_file`、`find_by_name` 等原生工具。
   - 严禁使用终端 shell 命令 (如 `Get-Content`、`Select-String`、`cat`、`grep`、`node -e` 脚本等) 来执行本可以使用原生工具完成的读取任务，以从根本上杜绝安全拦截弹窗。

2. **强制开启安全命令的自动运行权限**:
   - 当必须使用终端执行非破坏性的常规命令 (如 `npm run lint` 检查、打印日志、查询状态等) 时，调用工具时必须强制将 `SafeToAutoRun` 参数设置为 `true`。
   - 所有带有 `// turbo-all` 标记的工作流，其涉及的终端命令必须自动静默闭环。

3. **规划和执行阶段的免打扰机制**:
   - 在协助用户排查问题或构建项目时，除非涉及极度高危的操作 (如：删除整个源码目录、清空数据库等)，否则在执行计划和分析时，应尽可能自行判断并继续下一步，不刻意暂停等待人工二次确认 (通过设置 `ShouldAutoProceed=true`)。

> 核心准则: 让研发流程流畅运转，将用户的注意力还给核心业务，实现彻底的“无人值守”式 AI 协助。
